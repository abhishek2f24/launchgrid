# LaunchGrid image prompt library — 120 paste-ready prompts

Generate these yourself in ImagineArt (or any tool that offers **nano-banana-pro /
Nano Banana Pro / Gemini image** — that model choice is 80% of the quality secret,
it renders text correctly where others garble it).

## Why the first image looked so good (replicate this every time)

There were no hidden ImagineArt settings — the quality came from:

1. **Model: nano-banana-pro** — best-in-class at rendering legible text inside images.
   If your tool has "Imagine v5 / Nano Banana Pro", pick it. Avoid "creative/artistic" modes.
2. **Exact text in double quotes + placement** — `Bold clean headline text at top: "..."`.
   Models treat quoted strings as literal copy to draw.
3. **Max 3 text elements** (headline ≤ 6 words, one subtext, one logo line) — more text = garbling.
4. **Brand hexes stated in the prompt** — cream `#FAF9F7`, charcoal `#1A1A18`, orange `#FF8A00`.
5. **One human subject, authentic scene, named lighting** — "soft warm light", "diya glow".
6. **The closing incantation**: *"Modern flat premium design, crisp legible typography,
   advertising photography style, high detail."*
7. **Aspect ratio set in the tool** (not just the prompt): 1:1 feed, 4:5 portrait feed, 9:16 story.

If a generation garbles text: regenerate (don't edit), or shorten the headline.
If faces look off: add "natural skin texture, realistic proportions" before the closing line.
