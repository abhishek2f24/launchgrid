# LaunchGrid — Master Feature & Strategy Roadmap
> Goal: Become the #1 all-in-one platform for anyone starting a business in India with zero prior experience.
> Every item below is an implementation target. Priority: 🔴 Critical → 🟠 High → 🟡 Medium → 🟢 Nice-to-have

## 1. ANALYTICS & INTELLIGENCE (Merchant Dashboard)

### Built-in Store Analytics
- 🔴 Live visitor counter on dashboard (how many people on store right now) ✓
- 🔴 Today / 7-day / 30-day revenue, orders, AOV (Average Order Value) cards ✓
- 🔴 Traffic sources breakdown (direct, WhatsApp share, Google, social) ✓
- 🔴 Top products by revenue and by views ✓
- 🔴 Conversion funnel: Visitors → Product views → Cart adds → Checkout → Orders ✓
- 🟠 Hourly traffic heatmap (which hours get the most visitors)
- 🟠 Returning vs new visitor ratio
- 🟠 Bounce rate per page
- 🟠 Average session duration
- 🟠 Geographic breakdown of visitors (state/city level — India-specific)
- 🟡 Device breakdown (mobile vs desktop — critical for India where 90%+ is mobile)
- 🟡 Exit page report (where users drop off)

### Scroll & Behaviour Analytics (for merchant's store pages)
- 🟠 Scroll depth heatmap on product pages (how far users read)
- 🟠 Click heatmap (what areas of product page users click)
- 🟠 Session replay (watch recordings of real user sessions — Hotjar-style)
- 🟡 Rage click detection (where users click repeatedly out of frustration)
- 🟡 Dead click detection (clicks on non-clickable elements)
- 🟡 Form analytics (where users drop off in checkout)

### Business Intelligence
- 🟠 Revenue forecast based on last 30-day trend
- 🟠 Best day of week / best time of day to post promotions
- 🟡 Cohort analysis (which acquisition month retains best)
- 🟡 Customer Lifetime Value (LTV) per segment
- 🟡 Profit margin calculator per product (cost price vs sell price)
- 🟡 Inventory velocity report (how fast each product sells)

## 4. SEO TOOLS (For Merchant Stores)

### Technical SEO (Auto-handled by Platform)
- 🔴 Auto-generated XML sitemap for every store (updated when products added) ✓
- 🔴 robots.txt — correct config, no accidental indexing blocks ✓
- 🔴 Canonical URLs — prevent duplicate product content ✓
- 🔴 Structured data (Schema.org) auto-injected: Product, BreadcrumbList, Organization ✓
- 🔴 Open Graph + Twitter Card meta tags (for WhatsApp/social link previews) ✓
- 🔴 Page speed < 3s (Core Web Vitals — already critical ranking factor)
- 🟠 Image auto-optimization + lazy loading (WebP conversion on upload)
- 🟠 Auto-generated alt text for product images (AI-powered from product name)
- 🟡 Hreflang tags (for future multi-language support)

### On-Page SEO Controls (Merchant-editable)
- 🔴 Custom meta title + meta description per product page ✓
- 🔴 Custom meta title + description for store homepage ✓
- 🟠 Custom URL slugs for products (/buy-blue-saree-jaipur vs /product/12345)
- 🟠 Product tags / collections (for category SEO pages)
- 🟡 Merchant blog (each store can publish articles — SEO + content marketing)
- 🟡 FAQ section per product (auto-generates FAQPage schema)

### SEO Dashboard for Merchants
- 🟡 Keyword rank tracker (see where store ranks for top 5 keywords)
- 🟡 Google Search Console integration (clicks, impressions, CTR — one-click connect)
- 🟡 SEO health score per store with actionable fix list
